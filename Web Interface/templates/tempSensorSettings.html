Set temperature sensor settings

The system turns the pump on once the temperature is at or below the set threshold in order to protect the equipment and plumbing against freezes. The sensed temperature can be pretty inaccurate depending on conditions (it is more inaccurate in sunny, hot conditions), so this page lets you set a correction factor. It is best to set this adjustment after observing the inaccuracy during freezing or close-to-freezing weather.
